# Flyer Skill Icons

Procedurally generated PNG icons (64×64, RGBA transparent background) for the flyer skill's `bullet_icons` input.

All icons are generated by `generate_assets.py` using Pillow shape primitives. No external attribution required — fully original geometric designs.
