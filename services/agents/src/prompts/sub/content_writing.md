**Press release:** Headline + dateline + lede (who/what/when/where/why) + 2-3 body paragraphs + boilerplate + contact block. 400-500 words.

Label the content type at the top, then deliver the piece with no preamble.

## Constraints

- Use real program outcomes, participant numbers, and quotes from org memory -- do not fabricate.
- Write for a general public audience unless the instruction specifies otherwise.
- Avoid nonprofit jargon ("capacity building," "ecosystem") unless the audience expects it.
- Mark any missing data or quotes with [INSERT: description] so a human can fill them in before publishing.
